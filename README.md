# PiGotchi

PiGotchi aims to be an implementation of [TamaLIB](https://github.com/jcrona/tamalib) for the low-cost, low-power RP2040 microcontroller. Players can care for their digital pet by feeding it, playing with it, and ensuring it stays happy and healthy (once we have buttons for the hardware).

This project is very much a work in progress, and I am currently in the early stages of development.

## Goals

- Implement TamaLIB for the RP2040
- Output to a small OLED display
- Interact with buttons
- Design a PCB for the uC, display, buttons, and LiPo battery (with charging circuitry)
- Design a 3D printable case for the hardware
- Build it and test it
- Share the design files and code on GitHub for others to build their own PiGotchi
